# Scripts

This repository contains a collection of scripts used for various cybersecurity challenges, including Capture The Flag (CTF) competitions, Hack The Box (HTB), TryHackMe rooms, and CVE exploitation.

## Overview

- **CTF Scripts**: Tools and scripts designed to assist in solving CTF challenges.
- **Hack The Box Scripts**: Automation scripts for tasks and challenges on Hack The Box.
- **TryHackMe Scripts**: Scripts tailored for completing TryHackMe rooms and exercises.
- **CVE Exploitation Scripts**: Scripts for exploiting known vulnerabilities (CVE).

## Usage

To run the scripts, ensure you have the necessary dependencies installed. Execute the scripts using Python or Bash as appropriate.
